Qualidade de Dados

O que é Qualidade de Dados?

Qualidade de Dados (em inglês, Data Quality) refere-se à avaliação da "saúde" dos seus dados. É uma medida de quão bem os dados atendem aos requisitos de um determinado contexto, garantindo que sejam adequados para o uso pretendido. Em outras palavras, dados de qualidade são precisos, completos, consistentes, confiáveis e relevantes para o negócio.
